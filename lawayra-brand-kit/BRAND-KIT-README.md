## Typography
- **Primary Font:** Poppins (Google Fonts) — all weights included in /fonts/
  - Light (300), Regular (400), Medium (500), SemiBold (600), Bold (700), Italic
- **Script Font:** Twister — used for tagline accents only
- **Tagline:** "connect. heal. grow." (lowercase, with periods)

## Key Brand Elements
- CTA buttons: Amber (#E4A017), fully rounded (border-radius: 40px), white text
- Section backgrounds alternate: cream → olive → beige → cream
- Headings: lowercase, Poppins SemiBold/Bold
- Grain textures used as subtle overlays on hero images
- Photography style: warm, natural, analog film feel
